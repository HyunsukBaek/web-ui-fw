The Web UI Framework is a set of widgets and other components for jQuery Mobile.

To get started, use the search at the top of the page, view the full listing of entries, or browse by category from the sidebar.

This site provides API documentation for the widgets of the Web UI Framework.

The Web UI Framework supports jQuery Mobile 1.3.0 to 1.3.1, and jQuery 1.7.0 to 1.9.1.
